Sublime Text3安装配置

发表于 2018-07-22 | 分类于 配置 | | 次阅读

前言

一直习惯用Sublime Text编辑器，通过安装一些插件，Sublime Text的功能变得很强大，可以根据自己的需求安装插件，下面介绍几个很实用的插件。

一、Sublime下载安装

官网上直接下载sublime text3进行安装

二、插件安装

1. 安装Package Control

在安装插件之前需要先安装Package Control.安装方式是按住ctrl+shift+p打开插件安装弹框，输入关键词package install或者直接输入关键词pci选择package control: package install回车即可安装成功。

2. Color Theme主题配置

Sublime Text自带了几种主题，可通过Preferences->Color Scheme更换主题样式.另一种方式是安装主题插件，推荐Material Theme,安装方式：
Step1: command+shift+p打开package install控制台
Step2: 输入Material Theme回车，在安装完成后会提示你安装该主题配置的icon样式，选择安装。
Step3: 激活Material Theme,command+shift+p打开控制台，输入Material Active回车

3. emmet自动补全代码

emmet是一个HTML/CSS代码快速编写插件。安装完后，对于新的html文件，输入字符!，然后按Tab键会自动补全html基础结构。
Step1: ctrl+shift+p打开package install控制台
Step2: 输入emmet回车,下方会显示安装进度，安装完成后会显示说明文档。
Step3: 安装PyV8插件
一般情况下当安装完emmet插件后会自动安装PyV8插件，这种自动安装方式可能会失败，这时需要手动下载PyV8插件并安装。安装方式如下：
Step1: 下载PyV8插件，git地址
Step2: 拷贝到Sublime中
将下载的文件解压缩，如果是windows 64bit环境下找到Sublime Text3对应的pyv8-win64-p3文件，解压缩后将其中的_PyV8.pyd和PyV8.py两个文件拷贝到PyV8/windows-p3文件夹下(点击Preferences->Browse Packages新建PyV8/windows-p3文件夹)。
Step3: 安装PyV8插件，再次打开Sublime后需等待安装结束。

4. HTML-CSS-JS Prettify格式化代码

HTML-CSS-JS Prettify是一个代码格式化插件，其依赖于Node.js，所以需要安装Node.js.

Step1: 官网下载安装Node.js
Step2: Sublime中安装HTML-CSS-JS Prettify插件
Step3: 确认Node.js安装路径
在Sublime界面右键选择HTML-CSS-JS Prettify->Set node Path确认Node.js安装路径是否正确。
Step4: 格式化代码
鼠标右键HTML-CSS-JS Prettify->Pretty Code
