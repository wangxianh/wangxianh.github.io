高性能建站指南总结

发表于 2018-07-22 | 分类于 前端 |

前言

网站性能对前端工程师来说非常重要，书中总结了14条规则来提升网站性能，以下是自己的读书笔记。

1. 减少HTTP请求

  • 使用图片地图在一个图片上关联多个URL
  • 使用CSS Sprites将多幅图片合并为一幅单独的图片。使用background-image 和 background-position属性将HTML元素放置到背景图片中期望的位置。
  • 内联图片：通过使用data:URL模式可以在web页面中包含图片而不需要额外的http请求。
  • 合并脚本和样式表

2. 使用CDN(内容发布网络)

CDN是一组分布在不同地理位置的Web服务器。用于发布静态内容，包括图片、样式表、脚本和Flash。CDN是将源站内容分发至最接近用户的节点，使用户可以就近获取所需内容，提高用户访问的响应度和成功率。
CDN网络相当于是在用户和服务器之间增加Cache层，通过接管DNS的实现来将用户的请求引导到Cache上获得源服务器上的数据。

3. 添加Expires头

Web服务器使用Expires告诉客户端可以使用一个组件的缓存，直到指定的时间为止。
Expires缺点：

  • 要求服务器和客户端的时钟严格同步
  • 过期日期需要经常检查，一旦过期需要在服务器配置中提供一个新的日期

HTTP1.1中引入了Cache-Control来克服Expires的限制，Cache-Control使用max-age来指定组件缓存时间。如果两者同时使用，max-age会覆盖Expires.

4. 压缩组件

使用gzip压缩HTTP响应包来减少网络响应时间。压缩内容包括：HTML文档、脚本、样式表。
HTTP1.1开始，Web客户端可以通过HTTP请求中的Accept-Encoding来表示对压缩的支持.

Web服务器通过响应中的Content-Encoding来通知Web客户端。

5. 将样式表放在顶部

将样式表放在顶部的HEAD中可以避免白屏

6. 将脚本放在底部

7. 避免CSS表达式

8. 使用外部javascript和css

9. 减少DNS查找

10. 精简javascript

11. 避免重定向

12. 删除重复脚本

13. 配置ETag

14. 使Ajax可缓存

Sublime Text3安装配置

发表于 2018-07-22 | 分类于 前端 |

前言

一直习惯用Sublime Text编辑器，通过安装一些插件，Sublime Text的功能变得很强大，可以根据自己的需求安装插件，下面介绍几个很实用的插件。

一、Sublime下载安装

官网上直接下载sublime text3进行安装

二、插件安装

1. 安装Package Control

在安装插件之前需要先安装Package Control.安装方式是按住ctrl+shift+p打开插件安装弹框，输入关键词package install或者直接输入关键词pci选择package control: package install回车即可安装成功。

2. Color Theme主题配置

Sublime Text自带了几种主题，可通过Preferences->Color Scheme更换主题样式.另一种方式是安装主题插件，推荐Material Theme,安装方式：
Step1: command+shift+p打开package install控制台
Step2: 输入Material Theme回车，在安装完成后会提示你安装该主题配置的icon样式，选择安装。
Step3: 激活Material Theme,command+shift+p打开控制台，输入Material Active回车

3. emmet自动补全代码

emmet是一个HTML/CSS代码快速编写插件。安装完后，对于新的html文件，输入字符!，然后按Tab键会自动补全html基础结构。
Step1: ctrl+shift+p打开package install控制台
Step2: 输入emmet回车,下方会显示安装进度，安装完成后会显示说明文档。
Step3: 安装PyV8插件
一般情况下当安装完emmet插件后会自动安装PyV8插件，这种自动安装方式可能会失败，这时需要手动下载PyV8插件并安装。安装方式如下：
Step1: 下载PyV8插件，git地址
Step2: 拷贝到Sublime中
将下载的文件解压缩，如果是windows 64bit环境下找到Sublime Text3对应的pyv8-win64-p3文件，解压缩后将其中的_PyV8.pyd和PyV8.py两个文件拷贝到PyV8/windows-p3文件夹下(点击Preferences->Browse Packages新建PyV8/windows-p3文件夹)。
Step3: 安装PyV8插件，再次打开Sublime后需等待安装结束。

4. HTML-CSS-JS Prettify格式化代码

HTML-CSS-JS Prettify是一个代码格式化插件，其依赖于Node.js，所以需要安装Node.js.

Step1: 官网下载安装Node.js
Step2: Sublime中安装HTML-CSS-JS Prettify插件
Step3: 确认Node.js安装路径
在Sublime界面右键选择HTML-CSS-JS Prettify->Set node Path确认Node.js安装路径是否正确。
Step4: 格式化代码
鼠标右键HTML-CSS-JS Prettify->Pretty Code

5. Sidebar Enhancements浏览器打开html文件

Step1: 安装Sidebar Enhancements插件
Step2: 设置快捷键，打开Preferences->Key Bindings,在User中输入：

6. Color Highlight 颜色高亮

7. MarkdownExtended编辑markdown文件

8. MarkdownPreview浏览器打开markdown文件

9. PrettyJSON格式化json代码
